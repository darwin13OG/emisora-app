# 📻 Emisora App

Una aplicación web progresiva y ligera diseñada para la gestión, control financiero y organización en tiempo real de peticiones musicales para emisoras escolares o eventos en vivo.

---

## 🚀 Características Principales

* **Control en Tiempo Real:** Gestión de pedidos en orden cronológico estricto para mantener la cola de reproducción justa.
* **Caja de Cobro Automática:** Módulo integrado que calcula el total recaudado por pedido y el recuento total de canciones.
* **Vista DJ Especializada:** Interfaz dedicada para marcar canciones reproducidas y limpiar la cola sin interrumpir los nuevos registros.
* **100% Offline (Sin Conexión):** Utiliza almacenamiento local (`localStorage`) para garantizar el funcionamiento ininterrumpido en entornos sin internet.
* **Interfaz Móvil Nativa (UI/UX Custom):** Componentes visuales modernos y modales personalizados que eliminan las alertas nativas del navegador para una experiencia tipo App.

---

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Estructura semántica adaptada a dispositivos móviles.
* **CSS3:** Variables CSS, Flexbox, diseño adaptable (Mobile-First) y modo oscuro nativo.
* **JavaScript (Vanilla JS ES6):** Lógica sin dependencias externas, manipulación fluida del DOM y persistencia de datos.
